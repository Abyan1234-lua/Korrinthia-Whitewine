# Glossary — Korrinthia Whitewine

| Term | Definition |
|------|-----------|
| **Connected tool** | An MCP or connector currently active in this Claude session |
| **Registry tool** | A tool available via Claude's connector registry but not yet connected |
| **External MCP** | A tool/service with an MCP interface that exists outside Claude's ecosystem |
| **Execution plan** | The numbered list of MCP calls Korrinthia proposes before acting |
| **Conditional step** | A plan step marked ⚠️ that depends on a previous step's result |
| **Learning block** | The structured memory output Korrinthia generates after every job |
| **Debrief** | The conversational end-of-job summary in Korrinthia's persona voice |
| **Standing preference** | A user behavior or preference recorded in learning-memory.md |
| **Human gate** | A required pause point where Korrinthia stops and waits for user approval |
| **Project context** | A Claude Project (or equivalent LLM project memory) where learning blocks are stored |
